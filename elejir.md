# Elejir
Si tenemos una compararación a ralizar con más de tres opciones posibles, esta es la mejor solución;

```
ciudad = 'Quito'
elegir(ciudad)
  caso 0:
      escribir("i vale 0")
  caso 1:
      escribir("i vale 1")
  caso 2:
      escribir("i vale 2")
  defecto:
      escribir("i vale 100")
fin
```
