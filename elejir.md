# Elejir
Si tenemos una compararación a ralizar con más de tres opciones posibles, esta es la mejor solución;

```
ciudad = 'Quito'
elegir(ciudad)
  caso 'Quito':
      escribir("Estas en Peru")
  caso 1:
      escribir("Estas en Mexico")
  caso 2:
      escribir("i vale 2")
  defecto:
      escribir("i vale 100")
fin
```
