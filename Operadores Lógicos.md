# Operadores lógicos

```
v = verdadero
f = falso
```
### && y lógico (and)
Solo es verdadero cuando ambos miembros son verdaderos
```
escribir(f && f) #falso
escribir(v && f) #falso
escribir(f && v) #falso
escribir(v && v) #verdadero
```
### || y lógico (and)
Solo es verdadero cuando uno de los dos miembros es verdadero
```
escribir(f || f) #falso
escribir(v || f) #verdadero
escribir(f || v) #verdadero
escribir(v || v) #verdadero
```
