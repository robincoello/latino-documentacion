## Operadores:

A continuación se listan los operadores:

| Categoría | Operadores |
| :--- | :--- |
| Aritmético | `+, -, *, /, % (modulo), ^ (potencia)` |
| Lógico | && \(y\), \|\| \(o\) |
| Concatenación | `..` |
| Incremento, decremento | `++, --` \(valido solo en post\) |
| Relacional | `==, !=, <, >, <=, >=, ~= (regex)` |
| Asignación | `=` |
| Acceso a miembro | `.` |
| Operador ternario | `(condicion) ? opcion1 : opcion2` |
