# variables

Para definir una variable en "Latino" se lo hace de la siguiente manera:


### variable con texto
```
letra = 'a'
nombre = 'Latino'

```

### Variable con valor númerico
```
metros = 250.5
hora = 12
```

### Valor lógico, verdadero
```
var = verdadero
var = v
var = 1
```
