# variables

Para definir una variable en "Latino" se lo hace de la siguiente manera:


### variable con texto

```
var = 'a'
var = 'Latino'
var = 'Latino, lenguaje de programación en español'


```

### Variable con valor númerico
```
var = 250.5
var = 12
```

### Valor lógico, verdadero
```
var = verdadero
var = v
var = 1
```

### Valor lógico, falso
```
var = falso
var = f
var = 0
```
